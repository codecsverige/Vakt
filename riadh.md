## FakturaVakt – منظم الفواتير والاشتراكات (React Native)

### نظرة عامة
- تطبيق React Native ينشر على Google Play (ويمكن توسيعه لاحقًا إلى iOS).
- الهدف: مساعدة المقيمين في السويد على متابعة فواتيرهم واشتراكاتهم المتكررة وتجنب رسوم `påminnelse` أو التحويل إلى Inkasso.
- المستخدم يدير كل الدفعات في مكان واحد، مع أرشفة رقمية للفواتير الورقية وتذكيرات ذكية قبل الاستحقاق.

### الجمهور المستهدف
- المقيمون الجدد أو العائلات التي تتعامل مع فواتير متعددة (كهرباء، إنترنت، تأمين، اشتراكات بث).
- المستخدمون الذين يتحدثون السويدية أو العربية ويحتاجون واجهة بسيطة ثنائية اللغة.

### القيمة الأساسية
1. **إدخال وتتبع الفواتير**: تسجيل اسم الخدمة، المبلغ، تكرار الدفع، تاريخ الاستحقاق، ووضع علامة “مدفوع”.
2. **تنبيهات قبل الموعد**: إشعارات Push محلية قابلة للتخصيص قبل موعد السحب بعدة أيام.
3. **لوحة إحصاءات شهرية**: تعرض إجمالي الإنفاق الحالي، مقارنة بالشهر السابق، وأعلى التصنيفات استهلاكًا.

### ميزات مميزة
- **مسح رموز QR للفواتير**: قراءة QR السويدية (Bankgiro/Swish OCR)، تعبئة الحقول تلقائيًا، وأرشفة نسخة رقمية لتجنب ضياع الورق.
- **أرشيف المرفقات**: إمكانية التقاط صورة للفاتورة أو رفع PDF وربطها بالسجل.
- **مساعد طلب تمديد الدفع**: إنشاء رسالة جاهزة (سويدية/عربية/إنجليزية) لإرسالها عبر البريد أو SMS لطلب مهلة أو تقسيط من مزود الخدمة، مع روابط سريعة لمعلومات الاتصال.
- **دعم ثنائي اللغة**: تبديل فوري بين السويدية والعربية من الإعدادات.

### تجربة المستخدم
1. تسجيل حساب (محليًا أو عبر بريد إلكتروني) ثم اختيار اللغة.
2. زر “إضافة فاتورة/اشتراك” لملء الحقول الأساسية أو مسح رمز QR للفواتير الورقية.
3. التطبيق يضيف الإدخال إلى قائمة “الدفعات القادمة”، ويعرض إشعارًا قبل الاستحقاق.
4. بعد الدفع، ينقر المستخدم “تم الدفع” ليُنقل السجل إلى قسم السجل السابق ويحدّث الإحصاءات.
5. عند استقبال فاتورة تأخير (`påminnelse`)، يستخدم المستخدم “مساعد التمديد” لإرسال طلب رسمي بالمهلة.

### البنية التقنية
- **الواجهة**: React Native مع TypeScript، استخدم `React Navigation` و`Redux Toolkit` أو `Zustand` لإدارة الحالة.
- **التخزين**: قاعدة بيانات محلية عبر `SQLite` أو `MMKV`، مع خيار مزامنة إلى Google Drive أو Firebase Storage لاحقًا.
- **الإشعارات**: `react-native-push-notification` أو `Notifee` لتوليد تنبيهات مجدولة محليًا.
- **QR/OCR**: `react-native-camera` مع `ML Kit Vision` أو خدمة خارجية لقراءة رموز QR السويدية واستخراج بيانات الدفع.
- **التدويل**: `react-i18next` لإدارة ملفات اللغة (سويدي/عربي/إنجليزي).
- **التخزين السحابي الاختياري**: تكامل مع Google Drive API لحفظ النسخ الاحتياطية والمرفقات للمستخدمين البريميوم.

### الأمان والخصوصية
- تخزين البيانات مشفر محليًا (استخدام `react-native-encrypted-storage` أو تشفير SQLite).
- جميع الأذونات (كاميرا، إشعارات) تطلب عند الحاجة مع شرح واضح للمستخدم.
- سياسة خصوصية تحترم GDPR وتوضح أن البيانات لا تُباع لطرف ثالث.

### نموذج الربح
- **نسخة مجانية**: تعرض إعلانًا بنريًا خفيفًا في الشاشة الرئيسية وتقدم الوظائف الأساسية.
- **اشتراك بريميوم (2–3 € شهريًا أو خطة سنوية مخفضة)**:
  - إزالة الإعلانات.
  - تقارير PDF/Excel قابلة للتصدير.
  - نسخ احتياطي تلقائي إلى Google Drive.
  - تخصيص الألوان والتصنيفات، ومرفقات غير محدودة.
- **خدمات مستقبلية**: شراكات مع مزودي الكهرباء/الإنترنت للحصول على عمولة عند نقل المستخدمين إلى عروض أرخص.

### خطة العمل الأولية
1. **MVP (4–6 أسابيع)**: إدخال يدوي، قائمة الدفعات، إشعارات، لوحة إحصاءات، علامتي “مدفوع/إيقاف”، واجهة ثنائية اللغة.
2. **تحديث 1**: مسح QR وأرشفة المرفقات، تحسين تجربة التذكيرات.
3. **تحديث 2**: إطلاق الاشتراك البريميوم (تصدير PDF/Excel، نسخ احتياطي).
4. **تجربة المستخدمين**: حملات على TikTok/YouTube حول تجنب رسوم `påminnelse`, التعاون مع مجموعات فيسبوك للمقيمين الجدد.
5. **تحسين مستمر**: جمع تعليقات المستخدمين، إضافة مزايا مثل مراقبة البريد أو مقارنة العروض عند الحاجة.

### النشر والمتطلبات
- إنشاء حساب Google Play Console.
- إعداد لقطات شاشة (سويدية/عربية)، أيقونة، ووصف متجر يركز على منع الغرامات وتنظيم الاشتراكات.
- الالتزام بإرشادات Google بشأن الخصوصية (ملف Policy, روابط سياسة الخصوصية).
- تجهيز نسخة تجريبية عبر Closed Testing لجمع تعليقات مبكرة.

### الرؤية المستقبلية
- إضافة تكامل مع خدمات بريد رقمي مثل Kivra عندما يصبح ذلك ممكنًا.
- نشر نسخة iOS باستخدام نفس قاعدة الشفرة (React Native).
- بناء مجتمع مستخدمين يقدم نصائح للتوفير وإدارة المصاريف في السويد، ما يعزز الولاء والنمو العضوي.
